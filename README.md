## Задание 1







### Задача 1







### Текст задачи







#### Дерево содержит вещественные числа. Заменить отрицательные значения на
0, положительные — на 1.





### Алгоритм решения



######

#####Обход в глубину: Функция рекурсивно спускается от корня дерева к его самым нижним ветвям (Empty).Проверка условий: В каждом узле знак вещественного числа проверяется через конструкцию if-then-else.Замена значений: Положительные числа преобразуются в 1.0, отрицательные — в 0.0, а нулевые остаются без изменений.Рекурсивный спуск: Логика замены последовательно применяется к левому и правому потомкам текущего узла.Сборка структуры: На обратном пути рекурсии программа собирает и возвращает новое обновленное дерево из измененных элементов.

# Задание 2







###



### Текст задачи







#### Сколько элементов дерева содержит заданную цифру?







### Алгоритм решения







#### Запрос и чтение цифры: Программа считывает введенный пользователем символ (цифру от 0 до 9) из консоли.Запуск свертки (FoldTree): Дерево начинает обходиться в глубину (налево - центр - направо) с начальным значением счетчика, равным 0.Округление и приведение к строке: Значение текущего узла преобразуется в текст с округлением до двух знаков после запятой ("F2"), чтобы проверять только видимые на экране цифры.Поиск совпадения: Метод .Contains() проверяет, присутствует ли искомый символ-цифра в полученной строке.Обновление счетчика: Если цифра найдена, текущий счетчик увеличивается на 1, иначе — передается дальше без изменений.Вывод результата: После обхода всех узлов и ветвей итоговая сумма совпадений выводится на экран.
