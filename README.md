## Задание 1







### Задача 1







### Текст задачи







#### Дерево содержит вещественные числа. Заменить отрицательные значения на
0, положительные — на 1.





### Алгоритм решения



######

##### Алгоритм решения задачи: Создание структуры: 1. Описать дерево, где каждый узел хранит число, левую и правую ветки.Построение 2.(Вставка): Создать дерево поиска, добавляя меньшие числа влево, а большие — вправо. 3.Замена (Map): Обойти всё дерево и заменить положительные числа на 1.0, отрицательные на 0.0, а нули оставить на месте. 4.Вывод: Сгенерировать случайные числа, построить дерево, показать его структуру до и после замены в консоли.

# Задание 2







###



### Текст задачи







#### Сколько элементов дерева содержит заданную цифру? Fold







### Алгоритм решения







#### Создание структуры (Дерево): Создается рекурсивный тип данных Tree, который может быть либо пустым (Empty), либо узлом (Node) с числом и двумя ветками.Генерация данных: Функция generateTree рекурсивно строит дерево до заданной глубины, заполняя каждый узел случайными числами.Обход и сбор данных (Fold): Функция foldTree последовательно обходит все узлы дерева (от левых веток к корню и правым веткам), передавая накопленный результат (аккумулятор) от узла к узлу.Фильтрация по критерию: На каждом узле число переводится в строку с округлением до 2 знаков. Если в строке есть искомая цифра, счетчик-аккумулятор увеличивается на 1.Вывод результата: Программа красиво рисует структуру дерева в консоли и выводит итоговое количество найденных элементов.
